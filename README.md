# A Bioengineer's Guide to Molecular Biology

## This guide is currently ***under construction*** and will (hopefully) contain a wealth of knowledge to help the fledgling bioengineer get started in the wet lab

## Purpose

The goal of this GitHub repo is to have a living document that is easily editable by members of the scientific community. Currently, I only have one section of content on molecular cloning (specifically traditional restriction/ligation cloning and Golden Gate, with more cloning techniques on the way). I am currently writing some more articles on bacterial and mammalian cell culture.
